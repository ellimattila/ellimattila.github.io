Projects

Lapio Course Visuals

I designed a set of playful and clear illustrations for the Aalto University course CS-A0100: Tietokone työvälineenä. Each visual represents a course module.

LINUX
Represented by a playful keyboard illustration, this module teaches students how to navigate and utilize the Linux command line.

GIT
The collaborative “cooking pot” visual reflects how version control with Git helps manage and combine contributions in coding projects.

LATEX
A visual metaphor of a typesetting machine turning symbols into documents highlights the structured formatting power of LaTeX.

DATA
Depicted as a cityscape built from data blocks, this module covers managing and processing large datasets.

WEBPAGE
A visual representation of the course website.

OVERALL PATCH
Overall patch design for the course.

Graphics for Tietokilta’s 39th Annual Ball

This project includes the main graphics made for Tietokilta ry’s 39th annual ball, Muistinnollaus 100111. The materials cover both print and digital parts of the event. Not all graphics are shown here, but the key ones are included. The overall style is festive and elegant, using deep red and gold with decorative fonts to match the formal mood of the event.

Simplified Logo
A version of the event logo used in smaller formats, keeping the same elegant feel in a more minimal form.

Webpage
An information and registration webpage with a moving satin background done by rendering a ball.

Afterparty Food Ticket & Overall Patch
A ticket for the afterparty food and a patch for student overalls.

Song Booklet
Covers the evening’s program and dinner menu. The design connects with the rest of the event material and includes sponsor logos.

Namecard & Cocktail Card
A table namecard and a card for cocktail snacks.

Etiquette Guide
A quick guide designed to help guests feel confident at a formal student event.

Pixel Art for Aalto Game Jam

A set of animated pixel art sprites created for a small team project during Aalto Game Jam, under the theme “Space.” I designed this hamster character to bring charm and personality to the game’s universe. The animations were crafted in Aseprite.
