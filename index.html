Projects

Lapio Course Visuals

I designed a set of playful and clear illustrations for the Aalto University course CS-A0100: Tietokone työvälineenä. Each visual represents a course module.

LINUX
Represented by a playful keyboard illustration, this module teaches students how to navigate and utilize the Linux command line.

GIT
The collaborative “cooking pot” visual reflects how version control with Git helps manage and combine contributions in coding projects.

LATEX
A visual metaphor of a typesetting machine turning symbols into documents highlights the structured formatting power of LaTeX.

DATA
Depicted as a colorful cityscape built from data blocks, this module covers managing and processing large datasets.
